📐 Vector3f Class Documentation

Purpose: A 3D vector class for mathematical operations in 3D space with comprehensive geometric functionality.
Header: Core/Math/Vector3f.h
Dependencies: cmath, iostream, algorithm
Key Features: Public member access (x, y, z), comprehensive mathematical operations, utility functions for common 3D vector operations, and exception-safe division/normalization.
3D Vector Representation:
Vector3f(x, y, z)
x: right/left, y: up/down, z: forward/back

Usage Examples

Basic Vector Operations

Geometric Calculations

// Calculate surface normal from three points Vector3f p1(0, 0, 0), p2(1, 0, 0), p3(0, 1, 0); Vector3f edge1 = p2 - p1; Vector3f edge2 = p3 - p1; Vector3f normal = edge1.cross(edge2).normalized(); // Reflect ray off surface Vector3f incident(-1, -1, 0); Vector3f surfaceNormal(0, 1, 0); Vector3f reflected = incident.reflect(surfaceNormal); // Linear interpolation for animation Vector3f start(0, 0, 0); Vector3f end(10, 5, -3); Vector3f halfway = Vector3f::lerp(start, end, 0.5f); // Midpoint

Error Handling

try { Vector3f zero = Vector3f::zero(); Vector3f normalized = zero.normalized(); // Throws runtime_error } catch (const std::runtime_error& e) { std::cout << "Error: " << e.what() << std::endl; } try { Vector3f v(1, 2, 3); Vector3f result = v / 0.0f; // Throws invalid_argument } catch (const std::invalid_argument& e) { std::cout << "Error: " << e.what() << std::endl; }

Important Notes

  • Exception Safety: Division by zero and normalization of zero vectors throw exceptions
  • Epsilon Comparison: Equality uses epsilon (1e-6f) for floating-point comparison
  • Public Members: Direct access to x, y, z for performance-critical code
  • Coordinate System: Right-handed coordinate system (X=right, Y=up, Z=forward)
  • Performance: Use magnitudeSquared() when you don't need the actual magnitude
Exception Handling: Methods that can throw exceptions include normalize(), normalized(), angleTo(), projectOnto(), and division operators. Always handle these appropriately in production code.
