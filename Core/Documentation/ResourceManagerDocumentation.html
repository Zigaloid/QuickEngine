1. ?? Overview

The ResourceManager provides an asynchronous, thread-safe resource loading and management system for C++20 applications. It handles the complete lifecycle of resources from loading to finalization, with support for custom resource types and automatic memory management.

Key Features:
  • ? Asynchronous resource loading with dedicated worker thread
  • ? Thread-safe resource access and management
  • ? Template-based type-safe resource creation
  • ? Automatic resource deduplication (same path = same instance)
  • ? Complete resource lifecycle management
  • ? Integration with FileSystemManager
  • ? Extensible resource type system
  • ? Comprehensive monitoring and statistics

6. ?? Resource Lifecycle

Complete Lifecycle Flow

1. Request Phase (Any Thread)

  • Call RequestResource<T>(path)
  • Check if resource already exists
  • Create new resource if needed
  • Add to loading queue
  • Return shared_ptr immediately

2. Loading Phase (Worker Thread)

  • Initialize resource
  • Load data from file system
  • Move to finalization queue when complete

3. Finalization Phase (Main Thread)

  • Call UpdateFinalization() regularly
  • Process resources from finalization queue
  • Complete resource setup (GPU resources, etc.)

4. Usage Phase (Any Thread)

  • Use GetResource<T>(path) to access
  • Check IsResourceFinalized() for readiness
  • Shared ownership via shared_ptr

7. ?? Thread Safety

Thread-Safe Operations

Operation Thread Safety Synchronization
RequestResource() ? Thread-safe Mutex on resource map
GetResource() ? Thread-safe Mutex on resource map
IsResourceLoaded() ? Thread-safe Atomic state checks
UpdateFinalization() ?? Main thread only Queue synchronization
Resource state access ? Thread-safe Atomic member variables

Threading Model

Important Threading Rules:
  • Main Thread: Must call UpdateFinalization() regularly
  • Worker Thread: Handles Initialize() and Update()
  • Any Thread: Can call RequestResource() and GetResource()
  • Resource Access: Always use shared_ptr for thread-safe access

Basic Resource Loading

// Request resources (can be called from any thread) auto textureData = resourceManager.RequestResource("textures/grass.png"); auto soundData = resourceManager.RequestResource("sounds/footstep.wav"); auto modelData = resourceManager.RequestResource("models/tree.obj"); // Main game loop while (gameRunning) { // Update finalization on main thread (required!) resourceManager.UpdateFinalization(); // Check if resources are ready if (resourceManager.IsResourceFinalized("textures/grass.png")) { auto grass = resourceManager.GetResource("textures/grass.png"); if (grass && grass->IsFinalized()) { // Resource is ready to use std::cout << "Grass texture is ready!" << std::endl; std::cout << "Data size: " << grass->GetData().size() << " bytes" << std::endl; } } // Other game logic... std::this_thread::sleep_for(std::chrono::milliseconds(16)); // 60 FPS } // Cleanup resourceManager.Stop();

11. ?? Performance Considerations

Loading Performance

Performance Guidelines:
  • File I/O: Worker thread prevents main thread blocking
  • Memory Usage: Resources share instances for same paths
  • CPU Usage: Finalization spreads over multiple frames
  • Threading: Single worker thread prevents I/O contention

Memory Optimization

// Memory-efficient resource loading class MemoryEfficientResourceManager { private: ResourceSystem::ResourceManager& resourceManager_; size_t maxMemoryUsage_; std::atomic<size_t> currentMemoryUsage_{0}; public: MemoryEfficientResourceManager(ResourceSystem::ResourceManager& manager, size_t maxMemoryMB) : resourceManager_(manager), maxMemoryUsage_(maxMemoryMB * 1024 * 1024) {} template<typename T> std::shared_ptr<T> RequestResourceIfMemoryAvailable(const std::string& path) { // Estimate resource size (simplified) size_t estimatedSize = EstimateResourceSize(path); if (currentMemoryUsage_ + estimatedSize > maxMemoryUsage_) { std::cout << "Memory limit reached, cannot load: " << path << std::endl; return nullptr; } auto resource = resourceManager_.RequestResource<T>(path); currentMemoryUsage_ += estimatedSize; return resource; } void OnResourceUnloaded(const std::string& path) { size_t estimatedSize = EstimateResourceSize(path); currentMemoryUsage_ -= std::min(currentMemoryUsage_.load(), estimatedSize); } float GetMemoryUsagePercentage() const { return static_cast<float>(currentMemoryUsage_) / maxMemoryUsage_ * 100.0f; } private: size_t EstimateResourceSize(const std::string& path) { // Simple size estimation based on file extension std::string ext = path.substr(path.find_last_of('.') + 1); if (ext == "png" || ext == "jpg" || ext == "jpeg") { return 4 * 1024 * 1024; // 4MB for textures } else if (ext == "wav" || ext == "ogg") { return 2 * 1024 * 1024; // 2MB for audio } else if (ext == "obj" || ext == "fbx") { return 1 * 1024 * 1024; // 1MB for models } return 1024; // 1KB default } };
