📐 Matrix4f Class Documentation

Purpose: A 4x4 matrix class optimized for 3D graphics transformations with OpenGL-compatible column-major storage.
Header: Core/Math/Matrix4f.h
Dependencies: Vector3f, std::array, cmath
OpenGL Compatibility: Uses column-major storage format making it directly compatible with OpenGL. Internal array stores columns consecutively: m_matrix[0-3] = column 0, m_matrix[4-7] = column 1, etc.
4x4 Transformation Matrix Layout:
[ m00 m01 m02 tx ]
[ m10 m11 m12 ty ]
[ m20 m21 m22 tz ]
[ 0 0 0 1 ]
3x3 rotation/scale + translation vector + homogeneous row

Usage Examples

Basic Transformation

Important Notes

  • Matrix Order: A * B applies transformation B first, then A
  • Column-Major: Compatible with OpenGL, may need transpose for row-major APIs
  • Point vs Vector: Use transformPoint() for positions, transformVector() for directions
  • Angles: All rotation methods expect angles in radians
  • Singular Matrices: inverse() returns identity if determinant < 1e-6
