Topics:

Intro:

What I did?
	Attempted to build some core modules for a game engine?
	
Why I did it?
	To get a better handle on the capabilites of AI.

Process:
	Direct AI to implement a system.
	Minimal details on requirements.
	Compile validation.
	Request Unit Testing
	Iterate on test results.
	
Results:
	Demo
	Positive in the most part. 
	Very capable.
	
What is it good at?
	Refactoring.
	Documentation
	Testing, Unit test.
	Typeing :)
	
Wnat is it bad at?
	Multi system interactions.
	Novel designs.
	Consistency. (Maybe a style guide could help).

Conclusions:
	Don't ignore AI. 
	It's a very powerful programming tool.
	It allows you to focus on problem solving not just "Typeing".
	Will it replace programmers? Yes in some ways but it will not replace engineers/architects?
