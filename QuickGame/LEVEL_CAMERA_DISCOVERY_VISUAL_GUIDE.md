## Level Loading Process

### Initialize()

```
Start App
   ?
   ?? Setup Systems
   ?
   ?? Load Level File
   ?  ?? Create CLevelComponent
   ?
   ?? Initialize Level
   ?  ?? All components initialized
   ?
   ?? Search for Camera ?? NEW
   ?  ?? FindFirstCameraInHierarchy()
   ?
   ?? Attach Camera if Found ?? NEW
   ?  ?? SetActiveCamera(foundCamera)
   ?
   ?? Ready to Render
      ?? Uses camera for view/projection
```

### LoadLevel(path)

```
Load New Level
   ?
   ?? Release Old Level
   ?
   ?? Load New Level File
   ?
   ?? Initialize New Level
   ?
   ?? Search New Level ?? NEW
   ?
   ?? Attach New Camera ?? NEW
   ?  or Revert to Default
   ?
   ?? Ready
```
