## Rotation Speed Calculation

```
Input Value × Speed × Delta Time = Rotation Delta

Example: Frame at 60 FPS
?? frameTime = 1/60 = 0.0167 seconds
?? rightStickX = 0.8 (80% right)
?? yawSpeed = 2.0 rad/s
?
?? yawDelta = 0.8 × 2.0 × 0.0167
           = 0.0267 radians
           ? 1.53 degrees per frame

At 30 FPS:
?? frameTime = 1/30 = 0.0333 seconds
?? yawDelta = 0.8 × 2.0 × 0.0333
           = 0.0533 radians
           ? 3.06 degrees per frame

Both reach same rotation per second:
?? 60 FPS: 1.53° × 60 frames = 91.8° per second
?? 30 FPS: 3.06° × 30 frames = 91.8° per second ?

Frame-rate independent!
```

## Summary

Camera orbit input provides **standard third-person camera control** where:
- **Right Stick X** = Horizontal rotation (yaw)
- **Right Stick Y** = Vertical rotation (pitch)
- **Smooth, frame-rate independent** rotation
- **Configurable speeds** for game feel tuning
- **Optional pitch inversion** for player preference
